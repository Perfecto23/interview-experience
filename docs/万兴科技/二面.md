# 二面

> 时间：2025.11.12

## 一、 问题

### 面试官A (业务负责人)

1. 你有四年的前端经验，但我看到 ToC 端的经验相对偏少。未来如果加入我们团队（我们是 Web + AI 的应用服务），你在这块有什么思考和想法？比如，技术上你的擅长项是什么？希望发展的方向？或者希望团队提供哪些资源？
2. 你提到对 AI 前沿和新挑战感到兴奋。能举一两个具体的案例吗？介绍一下你在过往公司，在 AI 前沿这部分做得比其他同事更出色、更有创新或实践的例子。
3. 我们都在讲 AI 大模型，一种是专注做模型本身，一种是基于模型平台做应用和服务。你个人更看好哪个方向，或对哪个更感兴趣？
4. 你如何平衡“接触新的、有挑战性的东西”和“日复一日繁杂的工作”（比如需求响应、Bug 修复）？
5. 我们内部会有学习交流会，你愿意花个人时间（比如晚上或休息时间）来做一些小的交流分享吗？
6. 你上一份工作的强度大吗？我们是创新业务，Web+AI，市场变化快，产品迭代节奏非常快，比如上周六就干到晚上12点发版。这种情况会挺多。你怎么看这个事情？
7. 你觉得对于技术开发人员，哪些指标可以比较好地评估你的产出和绩效？

### 面试官B (技术负责人)

1. 你提到有一套学习方法论，平时通过哪些渠道去迭代更新你的知识？
2. 你平时有没有尝试做一些开源项目，或者在 GitHub 上发表一些自己的作品？
3. 你期望加入的新团队和他未来的领导应该具备哪些特点？

## 二、 反问

1. 这个岗位入职以后，主要是开发一个什么样的产品？

2. 刚刚说到绩效考核，想了解一下公司的绩效考核机制，分为哪些等级？大概的比例？以及晋升条件？

3. 公司的营收增长情况怎么样？
