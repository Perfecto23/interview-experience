# 一面

> 日期：2025.10.30
>
> JD上写这是一个40-80k的全栈岗位，我的简历是前端，他们希望给半年时间转换到全栈。

## 一、 问题

1. 你离职的原因是什么？
2. 你刚才提到用一个方法论就能拿A，可否介绍一下你主要的工作内容和负责的部分？以及你对职业规划是怎么想的？
3. 你觉得前公司这个情况是一种行业的现状，或者是一个赛道的现状，还是一种本公司的现状？
4. 那你现在对什么赛道比较感兴趣？
5. （关于全栈岗位）我们这个岗位，要的是你要理解我们的系统，然后你要能快速地修正这个系统中对用户体验不那么好的一种现象，不知道你对这个设计怎么考虑，或者说有没有信心？有没有兴趣？
6. 你觉得前端和 Vibe Coding，前端是不是最有可能被 Vibe Coding 取代？你怎么看待这个事？
7. 算法题：进阶版两数之和。给定一个包含重复元素的数组 nums 和一个目标值 target，找出数组中和为 target 的所有不重复的索引对。

## 二、 反问

1. 北美分布的公司分布很多吗？
2. 那北美市场，它主要由哪些岗位组成？
3. 北美也是有一些开发的，是吧？
4. 我之前有了解 MetaGPT 也是在开源社区里了解的，现在公司的产品只有这一个吗？
5. 现在公司的规模怎么样？就是人数。
6. 你公司老板和你是哪里人？
7. 那你相当于是一个创始人一样的角色吗？
