# 面委会一面

## 一、提问

1. 你选个最近做的项目吧。重点要跟我讲讲他的背景是什么？他要解决的难点是什么？然后你是怎么解决这些难点？或者说你觉得特别好的地方好不好？
2. 之前的一个构建和开发的方式是什么呀？Webpack 5 是吧？
3. 明白，就说他其实是痛点，大家，但是没有能力去解决他。
4. 诶这样，因为其实对于我而言 Rspack 这个也是一个新的东西，新的词啊。那你给我介绍一下它是什么东西啊？它比别的打包工具它的优点在哪里？以及它为什么有这个优点？好不好啊？技术原理是什么？对诶，你就相当于给我普及一下这个编码，因为我也不知道。
5. 但是其实 JS 我们也可以通过 Node 的一些机制把多线程打开。
6. 这里我打断一下，从技术原理上来说，如果是多线程或者说多进程它就很难做到内存安全，那这它 Rust 怎么做到？说我在多线程、多进程的情况下要做到内存安全。
7. 诶，稍等，我再打断一个，你刚才说一个叫模块联邦，对吧啊？你给我讲一下这个词是什么意思，好不好？
8. （迁移中）有遇到过，就是改了以后导致的问题。能不能举一个就是例子给我听一下？
9. Webpack Bundle Analyzer 是什么？
10. 诶，我觉得挺好的，尤其是你已经提了离职，然后还开始干这件事情，我觉得做得很好。那你为什么要离职啊？我感觉你对这个项目好像公司还是有感情的。
11. 那其实也Gap了一个多月了。那这一个多月、两个月，对，这一两个月在干嘛？

3. 嗯，其实我可能会好奇，就作为腾讯一家这样大厂，微信部门也是一个很老牌、很强大的部门。可能我会好奇，你们在用人上会不会去要求，比如说候选人3~5年具备哪些特质？比如说像可能偏思维方法论相关的。
