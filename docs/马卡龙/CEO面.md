# CEO面

> 日期：2025.11.01
>
> 一个AI Agent的创业公司，远程办公，感觉前景扑朔迷离，薪资不及预期，最终拒绝了。
>
> CEO非常年轻，留学回来直接创业的类型。

### 一、 问题

1. 你觉得你的最大的优点和最大的缺点分别是什么？

2. 这个缺点你后面是怎么去平衡的？

3. 你在接下来的两到三年里面，希望实现的职业发展目标是什么样的？希望自己成为什么样的一个人？

4. 这是你很想加入AI公司的原因吗？或者你选择一个AI公司以及一个创业公司，你是怎么思考的？

5. 在团队协作这方面，大家一般会怎么评价你？

6. 你对于下一份工作，还有什么想得到的东西，或者想锻炼到的东西？

7. 你觉得一个比较理想的工作日，从早到晚大致会做什么？
