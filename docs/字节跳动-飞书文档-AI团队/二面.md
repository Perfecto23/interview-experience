# 二面

> 日期：2025.11.13

## 一、 问题

1. 讲一下你之前参与过的一些具体的项目，找一个你觉得自己觉得比较有挑战的一个项目，你可以讲一下里面比较难的一些事情是什么？你可能是怎么去解决的？

2. 你们这一次的这个改造经验，最终比如说有办法比较低成本地复刻到其他的那些项目里面吗？

3. 中间有沉淀成一些具体的文档之类给到他们吗？有实际去推动，比如说其他的项目他们去接入之类的吗？

4. 那你走之前做这个事情，我看你还比较有热情，那为什么还是想要走？

5. 你想往前端一些领域去精通，有具体哪些比较感兴趣的领域吗？自己有去了解过吗？

6. 你作为之前（简历上写的）业务负责人，你会比较关注哪一些关键的指标？

7. 你怎么看待性能指标或者是质量指标这些东西？这几个指标它最终实际对业务的指标的一些影响？

8. 假设你现在也是这个业务的一个负责人，你怎么去判断这个事情？......你应该以什么样的 ROI 的一个评估的方式来决定这些事情的优先级？

9. 那最终怎么去量化讲这个事情？

10. 我看你简历里面你还做过一些全球化改造...大概有20万行，中间是怎么保证不出那个质量问题？

11. 质量只能人工校对吗？那个成本不会很高吗？......有，比如说现在再重新回头看这个事情有什么提效的方式吗？

12. 那当时为什么不去先对这个支持做一下提升，就做一下改造？

13. 后面（国际化）是通过那个 AI 相关的一些事情，然后做一些辅助校验，辅助生成词条 key...这个具体可以讲一下是怎么做的？

14. 看你简历里面下面还有一个事情，是一个 PDF 导出的问题...它具体是为什么很困难？

15. 用 Puppeteer 去做打印，有没有遇到什么 badcase？

16. 看你下面还有一个叫做一个安全 GPT，这个是一个什么样的工具？你在这个安全 GPT 里面主要负责的是做哪个？

17. 你有没有写过一些 Prompt，我怎么去调整那个Prompt，让那个模型回答得更准确？在这个安全 GPT 上这个项目里面有负责专门去写过，去调整那个 Prompt 吗？

18. （编程题）用 JS 实现一个并发控制器。

19. （算法题）给定一个整数数组 nums 和一个整数 target，以及一个整数 k，编写一个函数 kSum，找出数组中所有满足元素个数为 k 且元素之和等于 target 的不重复组合。例如，对于数组 nums = [1, 0, -1, 0, -2, 2]，target = 0，k = 4，函数应返回 [[-2, -1, 1, 2], [-2, 0, 0, 2], [-1, 0, 0, 1]]。

20. 你这个（kSum）算法它的复杂度多少？
