# 二面

> 日期：2025.11.17
>
> 二面整体也是非常的轻松。面试官是一个叫suyi的架构师，之前跟着天猪做eggjs的，最近回到影石。
>
> 这场面试也通过了，但是面试到这又被我单方面终止了。
>
> 影石的约面也是真的很慢，面完二面的时候已经到我面试大后期了。
>
> 手头Offer太多的时候，每天要花很多时间在HR身上（沟通、提交流水、背调材料....），就懒得继续面试了。

## 一、 问题

1. 简单介绍几个你觉得比较重要或者有代表性的项目。

2. 你们这边是怎么样一个分工？这十几个人都在这个项目里维护这100多个页面吗？

3. 选型的话是怎么来做？或者说你们怎么去协同的？

4. 我看这里面有一个架构引进，是从单体架构引进到 Monorepo（原话：Mario report），这边的选型你是怎么做的？考量点可以讲一下。

5. 这边的具体约束是指哪种？

6. 两个问题，第一个是用了 Monorepo，第二个是用了模块联邦。关于 Monorepo 这边选 PNPM Workspace 的原因是什么？社区还有其他方案，没有考虑过吗？

7. Webpack 5 迁移过程中遇到什么问题了？

8. 简历里提到全球化改造，达成300%营收增长。从技术视角来说，你觉得你做了哪些事情对达成这个目标有比较直接的作用？

9. 针对时区这个标准化，到底哪里比较困难？通常做法是后端上报用 UTC，前端拿数据转换。我想知道你觉得难点在哪？

10. 你通常怎么学习新技术？我看你的经历里没有写过 React。


## 二、 反问

1. 入职以后会负责什么业务？影石是不是有写 React Native (RN) 的？

2. 影石内部的前端基建怎么样？（组件库、脚手架、性能监控、CI/CD）

3. 方便问一下影石的前端团队规模及组织架构吗？

4. 在影石上班体验怎么样？
