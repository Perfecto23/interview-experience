# 一面

> 日期：2025.10.27
>
> 做AI语音的创业类型公司，可能更看重候选人意愿，以至于一面就是HR面。
>
> 技术面在线下进行，技术面结束后直接老板面。

## 一、问题

1. 你现在是一个离职状态是吗？
2. 行，那你这边介绍一下你这边的一个情况。
3. 是这样，你说你成长较快，但又没有收获，你指的“没有收获”是哪些方面？
4. 在这段工作里，你主要做哪些内容呢？
5. 你的前端工作主要做哪一块？你说有Web端、小程序、手机端，这三个端口你都有做过前端开发是吗？
6. 像跨端或跨浏览器的适配问题，你在前端开发中是怎么解决的？
7. 你在上家公司这几年，除了之前提到的问题，还主动发现过哪些能自己排查解决的问题？
8. 你说离职原因包括“公司环境不太好”，是内部原因还是其他外部原因？方便说一下吗？
9. 你这边是不是更专注在个人发展和技术提升上？
10. 后期找工作，你对公司类型规模、业务、岗位内容有没有想法？想找什么类型的工作？
11. 方便问一下，你在上家公司的薪酬是怎么样的？
12. 你看你这边有没有其他需要了解的问题？
13. 你们跟产品对接需求多吗？是你直接对接，还是由你leader对接？
14. 产品需求在上线前可能需要反复修改，你会排斥吗？我们产品即将上线，前端也有上线前反复修改的情况，你怎么看这一点？
15. 举个例子：国外消费者倾向“试用100分钟后直接付费”，国内消费者更关注“试用后怎么操作得折扣”，我们曾基于这一点修改产品开发（先做折扣流程，后因海外用户需求改回直接付费），这种需求变更你怎么看？
16. 我看你之前在上家公司做海外业务，你觉得服务海外产品和国内产品的前端技术开发，有哪些不一样？
17. 你现在是在哪里住？我们公司在龙岗黄阁坑，这个距离你可以接受吗？
18. 我们本来在招语音算法和前端推理加速岗，因计划接新项目，年底前预计扩张到30人左右，目前需补充约5人，但新项目还不确定，你了解这个情况吗？
19. 我们面试一般是三轮，若第二轮线下，会和第三轮合并（老板+技术一起聊），合并面会出题让你现场完成，你清楚吗？
20. 我们是先HR面再技术面（和大厂相反），老板更关注候选人实际情况和离职动机，这一点你了解吗？
21. 除了外地（如北京），第二轮面试一般是线下（曾报销厦门候选人交通费），线上线下有区别，你接受线下吗？
22. 你这边线下什么时间比较方便？

## 二、反问

1. 公司在哪上班？
2. 项目的开发流程是什么样的？比如从接需求到上线的流程。
3. 现在还要招几个人？
4. 三轮面试具体是哪三轮？第一轮是线上技术面吗？
5. 下一轮面试可能是线上还是线下？
